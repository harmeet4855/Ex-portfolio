Techniche'15 : Annual TechFest IIT Guwahati Website 

Project Description: Techniche is the annual-techno management festival of IIT Guwahati. As a core team member of Brand Communication and Creatives Team of Techniche, I was given a task to design,conceptualize and develop the website of techniche'15.

Project Status:Front-End Development Complete
Client: Techniche'15, IIT Guwahati

Project Duration: Feb'15
Design Responsibilities: Understanding categories of information  Wire-framing the mobile and bigger screen versions of the website according to set Information architecture

Technical Responsibilities:  Development of structure/layout & functionality of the pages using HTML5, CSS3, Bootstrap, Jquery etc.

Website: http://techniche.org/techniche15/
